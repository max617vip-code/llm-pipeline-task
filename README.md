# Задание 2: Автоматический пайплайн извлечения данных

Скрипт предназначен для автоматического извлечения характеристик товаров из текстовых описаний с помощью LLM (Groq API).

## Функционал
1. Чтение данных из `data/input.csv`.
2. Отправка описаний в модель `llama-3.3-70b-versatile`.
3. Парсинг JSON-ответа и принудительное добавление корректного ID.
4. Сохранение результата в `output.json`.

## Инструкция по запуску
1. Установите зависимости:
   `pip install pandas groq python-dotenv`
2. Создайте файл `.env` и добавьте туда ваш API-ключ:
   `GROQ_API_KEY=gsk_ВАШ_КЛЮЧ`
3. Запустите скрипт:
   `python main.py`

## Пример данных
Вход (input.csv):
id,description
1,Кроссовки Nike, синие, размер 42

Выход (output.json):
{
  "название": "Кроссовки",
  "бренд": "Nike",
  "цвет": "синий",
  "размер_или_характеристика": "размер 42",
  "id": 1
}
